### Теория графов:
####Алгоритм поиска Эйлерова цикла:
- Проверить связность графа (игнорируя изолированные вершины).
- Определить тип графа (ориентированный/неориентированный).
-Проверить условие существования цикла:
  - Для ориентированного: входящая степень = исходящей степени у всех вершин.
  - Для неориентированного: степень всех вершин чётна.
  - Использовать алгоритм Флёри/Иеронима: пройти по рёбрам, удаляя пройденные.
  - Вернуть цикл в обратном порядке.

####Алгоритм Крускала (минимальное остовное дерево):
- Создать систему непересекающихся множеств (DSU) для всех вершин.
- Отсортировать все рёбра по возрастанию веса.
- Последовательно добавлять рёбра в остов, если их концы принадлежат разным компонентам связности (объединять через DSU).
- Повторять, пока не будет добавлено V-1 рёбер.
